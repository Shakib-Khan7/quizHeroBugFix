Map:

Map function is used when we need a perform a task on an array and return the output.Example:"Doubling the elements of an array"

forEach:

It performs different tasks on the elements of the array but does not return anything.

Filter:

filter function is used when we put a condition on an array that returns the matched elements according to the condition.The returned value is an array

Find:

find function returns only the first matched element according to the condition from the array

Local storage:

local storage is used to store values or information,So that when we come back to the web page from the same browser,We can view what actions we did previously and continue from there

Session storage:

same as local storage but works for a certain period of time.If we close the tab and comeback we will not get previously used information

Cookies:

cookies are used to gather information on what the users frequently does on the website and use that information for better experience on further visit

Event Loop:

It handles asynchronous operations using events,callbacks,stack,queue
